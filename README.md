## Del ejercicio de CLAUDE CODE
### Realizar comentarios

Detalle y relación de las tareas solicitadas:

1.- [Tarea 1]
Utiliza Claude AI para copiar el código y pedirle que, sobre el mismo código, añada comentarios en cada línea, explicando la utilidad de cada elemento HTML y CSS. Puedes usar este prompt:
Para el código que te entrego, genera una nueva versión añadiendo comentarios didácticos en cada línea, explicando primero la funcionalidad de cada etiqueta HTML, y también para los estilos CSS y código de JavaScript. Documenta todo el código para estudiantes que están aprendiendo a programar, crea una documentación autoexplicativa que ayude a entender todo el código y en cada línea, añade también tu criterio y posibles alternativas proponiendo ejercicios alternativos para ampliar el código. Utiliza un lenguaje sencillo, añadiendo todos los elementos que sean necesarios para que el código pueda ser entendido o leído por terceras personas que quieren mantener y o ampliarlo.

2.- [Tarea 2](.pdf)
Pide que te genere un documento diferente sólo con la información explicativa del código, para añadirlo como documentación externa que pueda servir para un futuro.

3.- [Tarea 3](.md)+[Tarea_4](.md)+[Tarea_5](.md)
Una vez tengas los dos archivos, el fichero con el código completado con los comentarios y el fichero de texto externo, pídele que convierta este último fichero a formato md (Markdown). En este punto, para cada uno de los dos códigos, tendrás 3 ficheros (código fuente, código fuente con documentación y fichero md).

4.- [Tarea 6] 
Crea un repositorio en GitHub con el nombre “Calculadora_documentado” y “WebSemántica_documentado” y sube los tres ficheros para cada uno.
